## Быстрый старт

### Запуск (первый раз и всегда)
```bash
cd ~/.openclaw/workspace/atrani-booking-manager
./start.sh
```

### Основные команды
```bash
# Статус
./manage.sh status

# Логи (в реальном времени)
./manage.sh logs

# Синхронизация вручную
./manage.sh sync

# Показать IP для доступа из сети
./manage.sh ip

# Бэкап базы данных
./manage.sh backup

# Остановка
./manage.sh stop

# Запуск заново
./manage.sh start
```

### Проверка работы
```bash
# Health check
curl http://localhost:3002/health

# Список объектов
curl http://localhost:3002/api/properties | jq

# Все бронирования
curl http://localhost:3002/api/bookings | jq

# Дашборд (полная картина)
curl http://localhost:3002/api/dashboard | jq
```

## Troubleshooting

**Порты заняты?**
Измени в `docker-compose.yml` (секция `ports`) и перезапусти: `./manage.sh restart`

**Не работает из локальной сети?**
Проверь firewall на Mac (System Settings → Network → Firewall → разреши Docker)

**База пустая?**
Запусти синхронизацию: `./manage.sh sync`

**Логи показывают ошибки?**
Проверь: `./manage.sh logs booking-manager`

**Нужно пересобрать образы?**
После изменений в коде: `./manage.sh rebuild`
